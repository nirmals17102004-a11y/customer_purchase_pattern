## 🔄 Steps
1. **Data Loading**  
   Import dataset into Python using Pandas.

2. **Exploratory Data Analysis (EDA)**  
   - Summary statistics  
   - Data distributions  
   - Correlation analysis  

3. **Data Cleaning**  
   - Handle missing values  
   - Remove duplicates  
   - Standardize formats  

4. **SQL Queries**  
   - Load cleaned data into Oracle SQL Server  
   - Run queries for filtering, grouping, and aggregations  

5. **Dashboard Creation**  
   - Import SQL outputs into Power BI  
   - Build charts, KPIs, and interactive visuals  

6. **Report Writing**  
   - Document findings, methodology, and insights  

7. **Presentation**
